## 2. 문법 교육 방법론 (文法指導法)

### 2.1 상황 설정의 중요성 (状況設定)
> 문형 지도에서 학습자가 사용 장면을 구체적으로 이미지할 수 있도록 적절한 상황 설정이 필수

**적용 방법:**
- 말뿐 아니라 제스처, 그림 활용
- 실제 생활 상황 시뮬레이션
- 캐릭터 애니메이션으로 문맥 제공

### 2.2 직접법 (直接法)
> 일본어를 일본어로만 가르치는 방식, 다국적 학습자 환경에서 효과적

**핵심 원칙:**
- 교사는 "말을 너무 많이 하지 않음"
- 시각적/실연적 기법 활용
- 학습자 참여 중심

### 2.3 인풋-아웃풋 순환
```
[문법 설명] → [인풋 이해] → [아웃풋 연습]
      ↓              ↓              ↓
   규칙 제시    듣기/읽기 이해    말하기/쓰기
```

**인풋 이해 연습:**
- 텍스트 듣고 의미 파악
- 읽고 문법 포인트 확인

**아웃풋 연습 유형:**
1. **드릴 연습**: 정확성 향상
2. **타스크 연습**: 커뮤니케이션 목적

### 2.4 패턴 프랙티스 (Pattern Practice)
일본어 학교에서 자주 사용되는 연습법

| 유형 | 설명 | 예시 |
|-----|------|------|
| 반복 연습 | 같은 문장 반복 | "食べます" → "食べます" |
| 변형 연습 | 형태 변환 | "食べる" → "食べます" |
| 대입 연습 | 단어 교체 | "りんごを食べます" → "パンを食べます" |
| 확장 연습 | 문장 확장 | "食べます" → "朝ご飯を食べます" |
| 완성 연습 | 빈칸 채우기 | "___を食べます" |
| 응답 연습 | 질문-대답 | "何を食べますか?" → "パンを食べます" |

## 3. 어휘 교육 방법론 (語彙指導法)

### 3.1 레벨별 지도법

| 레벨 | 방법 |
|-----|------|
| 초급 | 어휘 학습 시간 별도 확보, 문법 수업 전 암기 |
| 중급+ | 수업 중 신출 어휘 즉시 교육 |

### 3.2 시각적/동작적 도입
- **그림 카드**: 구체적 명사
- **사진 자료**: 실제 상황
- **제스처 (G)**: 동사/형용사

**예시:**
> "とります" → 사진 찍는 제스처 → 직관적 이해

### 3.3 문맥 도입법
> 어휘는 단독보다 문맥 속에서 학습 시 장기 기억에 효과적

**원칙:**
- 문형이나 표현 안에서 도입
- 다른 단어와의 연결 의식
- 반복적 노출

### 3.4 학습자 동기 활용
> "모르겠다!"라고 곤란할 때 어휘 도입 → 학습 의욕 상승 → 정착률 향상

### 3.5 어휘 시트 활용
- 레벨별 맞춤 어휘 시트 작성
- 학습 시간 단축

### 3.6 활용 중심 학습
> 어휘 암기가 목표가 아님, 회화에서 사용할 수 있는 것이 목표

**방법:**
- 사용법과 배경 지식 보충
- 장면을 바꿔 활용
- 단문 만들기 활동
